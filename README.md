가천대학교 컴퓨터공학과 졸업인증 기록관리 웹사이트
====================================================
## Version 0.7


#### **디렉토리 설명**
1. /public 
2. /routes
3. /views
   - EJS 파일로 이루어져 있는 프론트
   - HTML, CSS, JS, bootstrap5
4. /settings
   - /database DB config 파일 존재

#### **계정상태**
현재 계정 상태에 따라 접속 가능한 페이지가 달라집니다.
1. 학생 -> /student
2. 조교 -> /assistant
3. 휴학 및 기타상태 -> 접속거부 및 안내
